# 🚀 T-Yap NG — Web Platform

## 🌍 Overview

T-Yap NG is a next-generation digital mobility and payment platform designed to power transportation, wallet services, and identity-based access across Nigeria.

This web application represents the public-facing interface of the T-Yap ecosystem — built for speed, accessibility, and scalability.

---

## 🎯 Project Goal

To deliver a fast, modern, and responsive web experience that introduces users to:

* Seamless transport solutions
* Secure wallet infrastructure
* Biometric-driven identity system
* Financial inclusion across Nigeria

---

## 🛠️ Tech Stack

* **Frontend Framework:** React (Vite)
* **Build Tool:** Vite
* **Styling:** CSS / Tailwind (if used)
* **Deployment:** DirectAdmin (DomainKing Hosting)

## ⚙️ Getting Started (Development)

### 1. Clone the repository

```bash
git clone https://github.com/your-repo/t-yap-web.git
cd t-yap-web
```

---

### 2. Install dependencies

```bash
npm install
```

---

### 3. Run development server

```bash
npm run dev
```

App will run on:

```bash
http://localhost:5173
```

---

## 🚀 Production Build

To generate production-ready files:

```bash
npm run build
```

This creates:

```bash
dist/
```

---

## 🌐 Deployment (IMPORTANT)

Only the **contents of the `dist` folder** should be deployed.

### Steps:

1. Zip the `dist` folder
2. Upload to server (`public_html`)
3. Extract files
4. Move contents to root:

```bash
public_html/index.html
public_html/assets/
```
